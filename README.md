# Folder Tree Visualizer

A minimal React + TypeScript + Vite project to visualize folder structures.

## Getting Started

### 1. Download the Project

You can download this project in one of the following ways:

- **Clone via Git:**

  ```sh
  git clone https://github.com/your-username/folder-tree-visualizer.git
  cd folder-tree-visualizer
  ```

- **Download ZIP:**
  - Click the green "Code" button on the repository page and select "Download ZIP".
  - Extract the ZIP and open the folder in your terminal or VS Code.

### 2. Install Dependencies

Make sure you have [Node.js](https://nodejs.org/) installed.

Install dependencies with:

```sh
npm install
```

### 3. Run the Development Server

Start the app in development mode:

```sh
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.

### 4. Build for Production

To build the app for production:

```sh
npm run build
```

The output will be in the `dist` folder.

```js
      // Alternatively, use this for stricter rules

      // Optionally, add this for stylistic rules

      parserOptions: {
        project: ["./tsconfig.node.json", "./tsconfig.app.json"],
        tsconfigRootDir: import.meta.dirname,
      },
      // other options...
    },
  },
]);
```
